
initially:

retrieve repository
git clone https://github.com/Yksu/tycoon.git
add shortcut
git remote add origin https://github.com/Yksu/tycoon.git

then to retrieve repo:
git pull

git add [EVERY MODIFIED FILE] (to see them, git status ; don't add ./project or ./target)
git commit -m "type your commit message here"
git push (origin master)


branches

see branches
git branch

create banch
git branch new_branch

go to an other branch (commit every file before going to an other branch!)
git checkout other_branch

merge other branch with master
git checkout master
git merge other_branch

delete branch
git branch -d other_branch (or -D to force)





remove .git folder
rm -rf ./.git
